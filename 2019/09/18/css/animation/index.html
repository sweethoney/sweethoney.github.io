发表于 2019-09-18 | 分类于 CSS ， CSS3

1、transition 过渡

transition过渡属性有以下几个特性：从：hover延伸出来的
1、过渡属性只有当鼠标经过时候才会作用
2、过渡属性没有中间的过程，动画是一步到位
3、过渡属性一次鼠标经过只可以发生一次，不能重复发生动画

2、animation 动画

相比transition更加具体，客观。可以一帧一帧设置值
