nvm、node、npm之间的区别

nvm：nodejs版本管理工具，一个 nvm 可以管理很多 node 版本和 npm 版本
node：在项目开发时所需要的代码库
npm：nodejs 包管理工具

nvm、node、npm之间的关系

nvm 管理 nodejs 和 npm 的版本。npm 可以管理 nodejs 的第三方插件。
